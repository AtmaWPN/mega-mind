# mega-mind
mastermind with variable colors and guess lengths
open Mastermind_Game.html to run the game
probably only works in Chrome or Safari

CONTROLS
	Menu:
		click on start or settings
	Settings:
		up/down arrows to navigate settings
		left/right arrows to change settings
		enter or click on back to return to menu
	Game:
		up/down arrows to select color
		left/right to change where the color will be placed
		enter to place color
		space to make a guess
		the amount of smaller red squares is how many colors in your guess are in the correct location
		the amount of smaller white squares is how many colors in your guess are misplaced
		click on back to return to menu
		click on reset to restart current game
